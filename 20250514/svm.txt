지도학습 알고리즘
분류와 회귀의 문제 해결
데이터를 가장 잘 나누는 결정경계 또는 초 평면을 찾는것이 목표 데이터가 선형적으로 분리되지 않는 경우(예: 데이터가 복잡하게 얽혀 있는경우), 커널 트릭을 사용해 문제를 해결
커널트릭:입력 자료를 고차원으로 변환하지 않고도 고차원 공간에서의 계산을 수행
 문제: 2D 평면에서 동그란 모양과 네모난 모양이 얽혀 있어 선으로 나눌 수 없음.
 커널 트릭: 데이터를 3D 공간으로 "던져서" 동그라미와 네모를 층층이 쌓이게 만듦.
 이제 평면(초평면)으로 쉽게 나눌 수 있음.
 데이터를 실제로 3D로 옮기지 않고, 커널 함수라는 "씨앗"을 사용해 3D에서의 유사성만 계산해 같은 결과를 얻음.
 마진 결정 경계와 서포트 벡터 사이의 거리
 결정 경계는 데이터 군으로 부터 최대한 멀리 떨어지는 게 좋다

 커널 기법
 커널:데이터 공간에서 고차원 공간으로의 매핑을 수행하는 함수
 선형, 다항식, 가우시안

가우시안 함수의 주요 특징
1. 종 모양의 곡선을 그리며, 대칭성.
2. 함수의 평균과 분산을 조절함으로써 곡선의 위치와 폭을 조절할 수 있다.
3. 중심으로부터 멀어질수록 함수값이 감소.
4. 확률 밀도 함수, 신호 처리, 이미지 처리, 머신러닝 등에서 활용.
5. SVM에서 데이터 간의 유사도를 측정하기 위해 사용되며,
 RBF 커널 함수의 폭을 조절하여 SVM 모델의 성능을 향상시킬 수 있다

 가우시안 커널
 가우시안 함수는 데이터 포인트 간의 거리에 따라 값이감소하는 형태를 가지며 가꾸운 포인트일수록 더 큰 가중치를 가진다

 감마는 커널의 매개변수로서 가꾸운 함수의 폭을 조절하는 역할

휴리스틱:
복잡한 문제를 해결하기 위해 경험적인 지침이나 근사적인 방법을 사용하여 문제에 대한 탐색이나 결정을 수행하는 방법
 휴리스틱은 최적해를 보장하지 않지만, 경험적인 노하우로 일반적으로 빠르게
실행되며 합리적인 결과를 얻을 수 있는 방법.
SVM은 모델 자체는 수학적, 커널이나 파라미터의 설정은 경험적 선택이 가능

감마는 커널 함수의 영향력을 조절하여 결정 경계의 유연성을 결정
작으면 과적합에 강함 큰 감마는 과적함에 약함
감마를 높이면 학습데이터에 많이 의존해서 오버피팅을 초래
낮추면 학습 데이터에 별로 의존하지 않고 결정 경계를 직선에 가깝게 긋게 된다 언더피팅이 발생가능
파라미터 
1. C는 모델의 정규화 강도와 오분류에 대한 페널티를 조절하는 파라미터.
2. 작은 C 값은 오분류를 허용하며, 큰 C 값은 오분류에 대해 더 엄격한 제약.
3. 작은 C 값은 결정 경계를 부드럽게 만들어 과적합에 강해지고,
 큰 C 값은 결정 경계를 더 정확하게 만들어 과적합에 약해진다.
* 파라미터 gamma는 커널 함수의 영향력을 조절하는 매개변수

SVR 은 회귀 마진 폭은 epilson 이라는 하이퍼 파라미터를 사용하여 조절한다

작은 값은 오차 범위를 좁게 만들어 정확한 예측을 시도하며, 큰 값은 오차 범위를 넓게 만들어 유연한 예측을 시도

하드마진: 이상치를 허용하지 않아 마진이 매우 작다 서포트 벡터와 결정 경계사이의 거리가 매우 좁다
이상치를 혀용하지 않는 기준으로 결정 경계를 버리면 오버피팅 방생

소프트 마진: 이상치들이 마진안에 포함되도록 기준을 잡는것. 서포트 벡터와 결정 경계 사이의 거리가 멀어졌다. 즉 마진이 커진다 언더 피팅 발생
scikit learn에서는 어느정도 오류를 허용할지 파라미터 c를 통해 지정 가능 c값이 클 수록 하드마진 작을수록 소프트 마진

장점 
효과적인 분류:선형 및 비선형 문제 모두에 대해 효과적인 분류를 제공.
마진 최대화:결정 경계와 데이터 포인트 사이의 마진을 최대화하므로, 일반화
능력이 뛰어나고 과적합을 줄일 수 있다.
커널 기법: 커널 기법을 사용하여 비선형 문제를 해결할 수 있다. 데이터를 고차원
공간으로 매핑하여 선형적으로 분리 가능한 문제로 변환하는 것.
이상치에 민감하지 않음: 서포트 벡터에 의존하기 때문에 이상치에 영향을 덜 받는다

SVM의 단점

계산 비용: 대규모 데이터셋에서 계산 비용이 크며, 메모리 사용량이 많을 수 있다.
파라미터 조정: 커널 타입, 커널 파라미터, 규제 파라미터 등의 조정이 필요. 이를
최적화하기 위해 교차 검증과 같은 추가 작업이 필요할 수 있다.
해석의 어려움: 결정 경계가 고차원 공간에서 계산되는 경우 해석이 어려울 수 있다.
